# Основы языка программирования Java

## Критерии оценки

Количество балов N вычисляется следующим образом:

N = 50 + P, где P — количество выполненных заданий.

| **Оценка** | **Кол-во выполненных заданий** |
| --- | --- |
| УДОВЛ | 10-19 |
| ХОР | 20-29 |
| ОТЛ | 30-50 |
